Operators

In the C programming language, operators are symbols that perform various operations on operands (variables, constants, or expressions) to produce a result. C provides a wide range of operators that can be classified into different categories based on their functionality.

It's important to understand the precedence and associativity rules of operators to write correct and meaningful expressions. Parentheses can also be used to explicitly control the order of evaluation.

Operators in C allow you to perform mathematical calculations, compare values, combine logical conditions, manipulate bits, assign values, and more. They are essential tools for performing various operations in C programs.

Arithmetic Operators
Operator Name Description Example
+ Addition Adds two values together x + y
- Subtraction Subtracts one value from another x - y
* Multiplication Multiplies two values together x * y
/ Division Divides one value from another x / y
% Modulo Gets the division remainder x % y
++ Increment Increments the value by 1 ++x
-- Decrement Decreases the value by 1 --x
Relational Operators
Operator Name Description Example
== Equal to Check if values are the same x == y
!= Not equal to Check if values a not the same x != y
> Greater than Check if one value is greater than the other x > y
< Less than Check if one value is less than the other x <
>= Greater than or equal to Check if one value is greater than or equal the other x >= y
<= Less than or equal to Check if one value is less than or equla to the other x <= y
Logical Operators
Operator Name Description Example
&& And True if both statements are true x && y
|| Or True if one statement is true x || y
! Not Opposite of the statement !x
Assignment Operators
Operator Name Description Example
= Simple assignment Set variable to value x = 5
+= Addition and assignment Add value to variable x += 5
-= Subtraction and assignment Subtract value from variable x -= 5
*= Multiplication and assignment Multiply variable by value x *= 5
/= Division and assignment Divide variable by value x /= 5
%= Modulo and assignment Set the variable to the remainder x %= 5
Bitwise Operators
Operator Name Description Example
&= Bitwise and Performs a bitwise AND operation between the corresponding bits of two operands. The result is 1 if both bits are 1, otherwise, the result is 0. x &= 5
|= Bitwise or Performs a bitwise OR operation between the corresponding bits of two operands. The result is 1 if at least one of the bits is 1, otherwise, the result is 0. x |= 5
^= Bitwise XOR performs a bitwise XOR operation between the corresponding bits of two operands. The result is 1 if the bits are different, otherwise, the result is 0. x ^= 5
~= Bitwise not performs a bitwise inversion of each bit. It flips each 0 to 1 and each 1 to 0. x ~= 5
>>= Right shift shifts the bits of the left operand to the right by a specified number of positions. Bits shifted beyond the rightmost position are discarded. x >>= 5
<<= Left shift shifts the bits of the left operand to the left by a specified number of positions. Bits shifted beyond the leftmost position are discarded, and zero bits are filled in from the right. x <<= 5
Other Operators
Operator Name Description Example
& Address-of Returns the memory address where the variable is stored. int* ptr = &x;
* Pointer dereference Used to access the value stored at a specific memory address pointed to by a pointer variable. int value = *ptr;
sizeof Sizeof Returns the size in bytes of a type or an expression. size_t size = sizeof(value);
, Comma Used to separate expressions. It evaluates each expression from left to right and returns the result of the rightmost expression. c = (a++, b++, a + b);
Conditional Operators

The conditional operator, also known as the ternary operator, is a unique operator in C that allows you to write a concise conditional expression. It provides a shorthand way to make a decision based on a condition.

condition ? expression1 : expression2
  • The condition is evaluated. If the condition is true (non-zero), then expression1 is evaluated and becomes the result of the entire expression.
  • If the condition is false (zero), then expression2 is evaluated and becomes the result of the entire expression.
