Data Types

In the C programming language, data types are used to specify the type of data that variables can hold. The choice of data type depends on the nature of the data and the operations that need to be performed on it. C provides several built-in data types, which can be broadly classified into the following categories:

Integer Types: Integer types represent whole numbers (positive, negative, or zero) without any fractional or decimal part. The size of integer types can vary across different systems. Common integer types in C include:

  • int: Used to store integers within a specified range (typically 32 bits).
  • short: Short integer type, usually smaller than int (typically 16 bits).
  • long: Long integer type, often larger than int (typically 32 or 64 bits).
  • char: Used to store small integers or characters (typically 8 bits).

Floating-Point Types: Floating-point types are used to represent real numbers with fractional parts. They can represent both positive and negative values. Common floating-point types in C include:

  • float: Single-precision floating-point type (typically 32 bits).
  • double: Double-precision floating-point type (typically 64 bits).

Character Types: Character types represent individual characters or small integers. In C, characters are enclosed in single quotes ('). Common character types include:

  • char: Used to store individual characters (typically 8 bits).

Void Type: The void type represents the absence of a value. It is often used to indicate that a function does not return a value or to define generic pointers.

Derived Types: C also allows the creation of derived types through structures, unions, and enumerations.

  • struct: A structure is a composite data type that groups together related data items of different types.
  • union: A union is a data type that allows storing different types of data in the same memory location. When a value is assigned to one member of a union, the other member may not hold a meaningful or predictable value. It's important to note that accessing the inactive member of a union can lead to incorrect results or unexpected behavior.
  • enum: An enumeration is a user-defined data type that consists of named values called enumerators.

#include <stdio.h>

// Define a struct to represent a point in 2D space
struct Point {
    int x;
    int y;
};

int main() {
    // Create an instance of the Point struct
    struct Point p1 = { 3, 5 };

    // Access and modify the struct members
    p1.x = 7;
    p1.y = 9;

    // Print the values of the struct members
    printf("Coordinates: (%d, %d)\n", p1.x, p1.y);

    return 0;
}


#include <stdio.h>

// Define a union to represent either an integer or a float
union Number {
    int intValue;
    float floatValue;
};

int main() {
    // Create an instance of the Number union
    union Number num;

    // Assign a value to the intValue member
    num.intValue = 42;
    printf("Integer value: %d\n", num.intValue);

    // Assign a value to the floatValue member
    num.floatValue = 3.14;
    printf("Float value: %.2f\n", num.floatValue);

    // Access the intValue member after assigning a float value
    printf("Integer value after assigning float: %d\n", num.intValue);

    return 0;
}


#include <stdio.h>

// Define an enum to represent weekdays
enum Weekday {
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
};

int main() {
    // Declare a variable of the Weekday enum type
    enum Weekday today;

    // Assign a value to the variable
    today = Wednesday;

    // Print the value of today
    printf("Today is ");

    switch (today) {
        case Monday:
            printf("Monday");
            break;
        case Tuesday:
            printf("Tuesday");
            break;
        case Wednesday:
            printf("Wednesday");
            break;
        case Thursday:
            printf("Thursday");
            break;
        case Friday:
            printf("Friday");
            break;
        case Saturday:
            printf("Saturday");
            break;
        case Sunday:
            printf("Sunday");
            break;
    }

    printf("\n");

    return 0;
}
