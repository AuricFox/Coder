Data Types
Type Description Examples
Integer (int) Numerical data type composed of whole numbers -5, 0, 123
Long (long integer) Integers with a large range set with signed 32-bit numbers ranging from -2,147,483,648 to 2,147,483,647 and unsigned 32-bit numbers ranging from 0 to 4,294,967,295 0, 1234567
Short (short integer) Integers with a small range set with signed 32-bit numbers ranging from -32,767 to 32,767 and unsigned 32-bit numbers ranging from 0 to 65,535 -5, 0, 123
Floating Point (float) Numberic data type used for numbers with decimals or fractions -5.5, 0.1, 123.5
Double Data type used for higher precision decimal numbers that has a 64-bit floating point -5.421, 0, 3.141
Date Data type that stores a calendar date as YYYY-MM-DD format 2023-01-27
Character (char) Data type that stores a single letter, number, or special symbol. A, b, 6, +
String (str or VARCHAR) Data type that stores a sequence of characters, numbers, or symbols MyString, b4life, h!4veryOne
Boolean (bool) Logical data type used for true or false values 0 (false), 1 (true)
Nothing (NULL or void) Data type containing no value NULL, void, NaN
