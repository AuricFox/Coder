Integer Types
In some programming languages the size of an integer that is stored can be set. Integers can also be unsigned which only allows positive values to be stored. The size of an integer data type is compiler-dependent. A 16-bit system gives an output of an int as 2 bytes, and a 32-bit system gives an output of an int as 2 bytes and 4 bytes.
