Date Types
There are many ways to format dates. Since dates are not primative data types, most programming languages require a module or library to be imported to set dates. The formatting depends on the module being imported with some common formats below.
Date Types Format Range
date YYYY-MM-DD 0001-01-01 to 9999-12-31
smalldatetime YYYY-MM-DD hh:mm:ss 1900-01-01 to 2079-06-06
datetime YYYY-MM-DD hh:mm:ss[.nnn] 1753-01-01 to 9999-12-31
datetime2 YYYY-MM-DD hh:mm:ss[.nnnnnnn] 0001-01-01 00:00:00.0000000 to 9999-12-31 23:59:59.9999999
