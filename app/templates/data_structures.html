Linear Data Structures
Data structures organized in a sequential or linear arrangement where each element is attached to elements before and after it. These structures can both be static (fixed memory size) or dynamic (memory size can change).
Array Linked List Queue Stack Hash Table
Trees
A non-linear data structure that resembles the shape of trees in real life. Unlike linear structures, trees can have multiple branches (children) and only one previous element (parent).
Binary Search Tree B-Tree Red-Black Tree AVL Tree
Graphs
Another non-linear data structure just like trees but can have multiple parent and child elements. These connections can also form cycles where the pathways will lead back to the original or starting element.
Undirected Graph Directed Graph Weighted Graph
