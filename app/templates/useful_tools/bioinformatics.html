Counting Codons
Count the number of codon contained within a genetic sequence. Every three characters of a string is a codon. This tool takes in a fna or fastq file containing genetic sequences and counts the total number of codons within the file.

Phylogeny Trees
Create a phylogeny tree with a collection of genetic sequences to see where the sequences diverged from each other. This tool takes in a fna or fastq file, compares the sequences, and creates a phylogeny tree.

Genetic Assembly
Assemble genetic fragments from sequence reads generated from lab results of DNA collection. This tool takes in a fna or fastq file, breaks the reads into codons, creates a de Bruijn graph to connect the codons, and assembles them into contigs.
