Structures

Structures in C allow you to define a custom data type that groups together variables of different types under a single name. Each variable within a structure is called a member or a field. Structures are useful for representing real-world entities with multiple attributes. 


            struct structName {
                dataType member1;
                dataType member2;
                // Additional members...
            };

Anonymous structures are structures without a tag name. They are typically used when you need to define a structure type inline without giving it a specific name. 


                struct {
                    dataType member1;
                    dataType member2;
                    // Additional members...
                } variableName;

The keyword typedef in C allows you to create aliases or alternative names for existing data types, including structures. It is particularly useful for making code more readable and manageable, especially when dealing with complex data types or when you want to abstract implementation details. 


            typedef struct structName {
                dataType member1;
                dataType member2;
                // Additional members...
            } StructType;
Accessing and Modifying Struct Members (Without Pointers)

Structure members are accessed using the (.) operater and assigned using an assignment (=) operator for non-string variables and the strcpy function for string variables. 


            struct Person {
                char name[50];
                int age;
            };
            
            int main() {
                // Declare a variable of type "Person"
                struct Person person1;
                
                // Accessing and assigning values to structure members
                strcpy(person1.name, "John");
                person1.age = 30;
                
                // Printing structure members
                printf("Name: %s\n", person1.name);
                printf("Age: %d\n", person1.age);
                
                return 0;
            }
Accessing and Modifying Struct Members (With Pointers)

Structure members are accessed using the (->) operater when working with pointers to structures. They are still assigned a value the same way when working with the (.) operater. 


            struct Person {
                char name[50];
                int age;
            };
            
            int main() {
                // Declare a pointer to a structure of type "Person"
                struct Person *ptrPerson;
                
                // Allocate memory for the structure using dynamic memory allocation
                ptrPerson = (struct Person *)malloc(sizeof(struct Person));
                
                // Check if memory allocation is successful
                if (ptrPerson == NULL) {
                    printf("Memory allocation failed\n");
                    return 1; // Exit the program with an error code
                }
                
                // Accessing and assigning values to structure members using '->'
                strcpy(ptrPerson->name, "John");
                ptrPerson->age = 30;
                
                // Printing structure members
                printf("Name: %s\n", ptrPerson->name);
                printf("Age: %d\n", ptrPerson->age);
                
                // Free dynamically allocated memory
                free(ptrPerson);
                
                return 0;
            }
