Switch Case

In C, the switch statement provides another way to perform conditional branching based on the value of an expression. It allows you to execute different blocks of code depending on the value of a variable or an expression. 


            switch (expression) {
                case x:
                    // Code Block
                case y:
                    // Code Block
                case z:
                    // Code Block
                default:
                    // Code BlocK
            }

Expression: The expression inside the switch statement is evaluated, and the value is compared with the constants specified in the case labels.

Case Labels: Each case label specifies a constant value that the expression is compared against. If the value of the expression matches a case label, the corresponding block of code is executed.

Break Statement: After executing the code for a particular case, it's important to include a break statement to exit the switch statement. Without a break statement, control will fall through to the next case, leading to unintended behavior.

Default Case: The default case is optional and is executed when the value of the expression doesn't match any of the specified cases. It's typically used as a catch-all case.

Fall-through Behavior: Unlike some other programming languages, C does not automatically exit the switch statement after a case is executed. If a break statement is not included after a case, control will continue to the next case, leading to a "fall-through" behavior. This can be intentionally used in some cases but should be used judiciously to avoid unintended behavior.

Supported Data Types: The expression inside a switch statement can be of integral types (int, char, short, long, etc.) or an enumeration type (enum). It does not support floating-point types or strings.

Nested Switch Statements: You can nest switch statements within other switch statements to handle more complex scenarios, but it's important to keep the code readable and maintainable.

#include <stdio.h>

int main() {
    int choice = 2;

    switch (choice) {
        case 1:
            printf("You chose option 1.\n");
            break;
        case 2:
            printf("You chose option 2.\n");
            break;
        case 3:
            printf("You chose option 3.\n");
            break;
        default:
            printf("Invalid choice.\n");
    }

    return 0;
}
