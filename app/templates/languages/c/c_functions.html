Functions

Functions are a fundamental concept in programming that allow you to encapsulate a set of instructions into a reusable block of code. 


            returnType functionName(parameter1Type parameter1, parameter2Type parameter2, ...) {
                // Function body
            }
Function Call

To use a function, you call it by its name, providing arguments if necessary. 


            // Just call function
            fuctionName(argument1, argument2, ...);

            // Get return result from function call
            dataType variableName = fuctionName(argument1, argument2, ...);
Return Statement

The return statement is used inside a function to return a value to the caller. It specifies the value to be returned (if any) and ends the function's execution. 


            return expression;
Return Type

The return type of a function specifies the type of value that the function returns to the caller. It can be any valid C data type, including built-in types (int, float, char, etc.) or user-defined types (structs, enums). If a function does not return any value, its return type is 'void'. 


            // Return an integer sum of a and b
            int add(int a, int b) {
                return a + b;
            }
Parameters

Parameters are variables declared in the function's parameter list. They receive values from the caller when the function is called. Inside the function, parameters act as local variables, initialized with the values passed from the caller. 


            // Return the sum of the parameters a and b
            int add(int a, int b) {
                return a + b;
            }
Arguments

Arguments are the values passed to a function when it is called. They correspond to the parameters declared in the function's parameter list. 


            // Add the arguments 7 and 8 to get the sum
            int sum = add(7, 8);

#include <stdio.h>

// Function definition
void modifyValue(int *ptr) {
    // Dereferencing the pointer to modify the value
    *ptr = 20;
}

int main() {
    int x = 10;
    printf("Before calling the function: %d\n", x);
    
    // Passing the address of x to the function
    modifyValue(&x);
    
    printf("After calling the function: %d\n", x);
    return 0;
}
