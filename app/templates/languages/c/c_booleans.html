Booleans

In C programming, a boolean is a data type that represents true or false values. However, C does not have a built-in boolean type like some other programming languages such as Python or Java. Instead, boolean values are typically represented using integers, where 0 represents false and any non-zero value represents true.

You can define boolean values using the standard C library, <stdbool.h>, which provides the bool type and two constants, true and false.

#include <stdio.h>

int main() {
    bool myBool = 0;

    if (myBool) {
        printf("The boolean value is true!\n");
    } else {
        printf("The boolean value is false!\n");
    }

    return 0;
}


#include <stdio.h>
#include <stdbool.h>

int main() {
    bool myBool = true;

    if (myBool) {
        printf("The boolean value is true!\n");
    } else {
        printf("The boolean value is false!\n");
    }

    return 0;
}
