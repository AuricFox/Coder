Conditionals

In C, conditional statements are used to control the flow of execution based on certain conditions. The primary conditional statements in C are the if, else if, and else statements. These statements allow you to execute different blocks of code depending on whether certain conditions are true or false.

Syntax

            if (condition1) {
                // Code Block
            } else if (condition2) {
                // Code Block
            } else {
                // Code Block
            }

if Statement: The if statement allows you to execute a block of code if a specified condition is true. If the condition inside the parentheses evaluates to true, the code block enclosed within curly braces is executed. If the condition is false, the code block is skipped.

else if Statement: The else if statement allows you to specify additional conditions to test if the preceding if condition is false. If condition1 is false, condition2 is evaluated. If condition2 is true, the corresponding code block is executed. You can have multiple else if blocks to test multiple conditions sequentially.

else Statement: The else statement is used to specify a block of code that should be executed if none of the preceding conditions are true. It is optional and always comes after the if or else if blocks. If none of the conditions specified in the if or else if statements are true, the code block following the else statement is executed.

#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("%d is positive.\n", num);
    } else if (num < 0) {
        printf("%d is negative.\n", num);
    } else {
        printf("%d is zero.\n", num);
    }

    return 0;
}


#include <stdio.h>

int main() {
    int num = -25;

    if (num > 0) {
        printf("%d is positive.\n", num);
    } else if (num < 0) {
        printf("%d is negative.\n", num);
    } else {
        printf("%d is zero.\n", num);
    }

    return 0;
}
