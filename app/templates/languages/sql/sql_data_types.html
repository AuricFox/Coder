Data Types

In SQL, data types (also known as variable types or simply types) define the type of data that can be stored in a column of a table or a variable in a stored procedure or function. Different database management systems (DBMS) support different data types, but there are common types that are widely supported across most SQL implementations.

Integer Types

INT, INTEGER: A whole number without a fractional component.

SMALLINT: A small integer, typically with a smaller range of values than INT.

BIGINT: A large integer, capable of storing larger values than INT.

Numeric Types

DECIMAL(precision, scale), NUMERIC(precision, scale): Exact numeric data type with a fixed precision and scale.

FLOAT, REAL, DOUBLE: Approximate numeric data types with floating-point precision.

Character and String Types

CHAR(n): Fixed-length character string with a specified length n.

VARCHAR(n), VARCHAR2(n): Variable-length character string with a maximum length n.

TEXT: Variable-length character string with no specified maximum length.

NCHAR(n): Fixed-length Unicode character string.

NVARCHAR(n): Variable-length Unicode character string with a maximum length n.

Date and Time Types

DATE: Date without a time component (YYYY-MM-DD).

TIME: Time without a date component (HH:MM:SS).

DATETIME, TIMESTAMP: Date and time combined (YYYY-MM-DD HH:MM:SS).

INTERVAL: Time interval.

Boolean Type

BOOLEAN, BOOL: Boolean values representing true or false.

Binary Types

BINARY(n): Fixed-length binary data with a specified length n.

VARBINARY(n): Variable-length binary data with a maximum length n.

BLOB: Binary large object for storing large binary data.

Other Types

ARRAY: Collection of values of the same type.

JSON: Stores JSON data.

XML: Stores XML data.

UUID: Universally unique identifier.

GEOMETRY, GEOGRAPHY: Spatial data types for storing geometric and geographical data.
